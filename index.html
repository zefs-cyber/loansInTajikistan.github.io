Finding Loans in Tajikistan

What is it?

This project aims to provide a comprehensive and user-friendly solution for customers in Tajikistan who are seeking to make informed decisions about their financial options by offering a sophisticated platform that enables them to easily compare and evaluate the diverse range of loans available from different banks operating within the country. By leveraging advanced technology, data analysis, and intuitive interfaces, the project strives to empower individuals with the necessary tools and information to navigate the complex landscape of loan offerings, thereby facilitating financial literacy and enabling individuals to make optimal choices based on their unique needs and preferences.

How to use?

  • Loan Search and Comparison: The project provides a comprehensive platform where users can search and compare various loan options available in Tajikistan. It offers an extensive database of loans from different banks, allowing users to explore and evaluate the terms, interest rates, repayment options, and eligibility criteria of each loan.
  • Real-Time Exchange Rate Information: In addition to loan search functionality, the project also offers real-time exchange rate information. Users can access the latest exchange rates, enabling them to make informed decisions regarding loans and currency conversions. This feature adds value by providing up-to-date financial data for users considering loans with foreign currency components or planning international transactions.
  • Bank Integration and Application Process: The project streamlines the loan application process by integrating with multiple banks in Tajikistan. Users can conveniently apply for loans directly through the platform, eliminating the need to visit individual bank websites or branches. The project simplifies the application procedure, allowing users to submit their loan applications electronically and track their progress.

Homepage

To start working with the bot user needs to send "/start" command or click on "Start" button

Selecting Language

User needs to select one of three languages:

  • Tajik language
  • Russian language
  • English language

Selecting Actions

In this step user has 3 steps to choose:

  • Help With Loan
  • Exchange Rates
  • Apply for Loan

Help With Loan

Users can specify their loan requirements, and the system will generate a curated list of banks that can potentially fulfill the user's desired loan criteria.

Exchange Rates

Users can see realtime exhange rates from National Bank of Tajikistan

Apply for Loan

After answering questions about their desired loan, users can apply for loan in 3 banks across Tajikistan and after that bank representatives will contact them

Help With Loan

Here user can search for banks which can give to user loan

First user needs to answer queestions about:

  1. Purpose of loan
  2. Currency
  3. Amount
  4. Duration

After answering all the queestions the bot will send results in the form of table where user can see interest rates of banks for his loan and link to the bank with lowest interest rates

Exchange Rates

Here user will answer to 3 questions:

  1. Which currency do you want to change?
  2. To which Currency do you want to change?
  3. How much do you want to change?
Note: All currencies must be written in short formats. Ex: TJS, USD, RUB

Apply For Loan

This action is prety much similar to "Help With Loan", but here we have 3 additional inputs, which are necessary to apply for Loan

Inputs:

  1. Select Bank
  2. Name Surname
  3. Phone number(without +992)
  4. Photo of National Passport
  5. Purpose of loan
  6. Currency of loan
  7. Amount of money
  8. Duration of loan

After system collects all required information it will send request to the selected bank
